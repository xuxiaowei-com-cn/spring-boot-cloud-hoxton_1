# Passport（服务模块）

## dependencies

- Developer Tools
    - Lombok
    - Spring Configuration Processor

- Web
    - Spring Web
        - 提供 Web 服务

- Modules
    - redis
        - 提供 Redis、Session共享 统一配置
    - mybatis
        - 提供持久层统一配置
    - client-utils
        - 作为配置服务器的客户端所需依赖
    - oauth2-utils
        - 作为需要权限的模块所需依赖
