# mybatis（MyBatis 配置）

- 作为普通 jar 包（非 Spring Boot），供其他模块引用，不需要启动

## dependencies

- Developer Tools
    - Lombok
    - Spring Configuration Processor

- Web
    - Spring Web

- SQL
    - MyBatis Framework
    - MS SQL Server Driver
    - MySQL Driver
    - Oracle Driver

- I/O
    - Validation

- Other
    - fastjson
    - mybatis-plus-boot-starter
    - mybatis-plus-generator
    - dynamic-datasource-spring-boot-starter
    - velocity
    - p6spy
    - seata-spring-boot-starter
        - druid
            - 已排除
