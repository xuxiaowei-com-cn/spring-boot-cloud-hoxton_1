# cloud-admin-vue

## 创建项目

~~~
npm i -g @vue/cli
npm i -g @vitejs/create-app
npm init @vitejs/app cloud-admin-vue --template vue
~~~

## devDependencies

- sass
    - [TAONPM](https://developer.aliyun.com/mirror/npm/package/sass)
    - [GitHub](https://github.com/sass/dart-sass)

~~~
npm i -D sass
~~~
